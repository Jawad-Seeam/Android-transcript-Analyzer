## 🆘 Emergency Reference

### "App won't start"
1. Check STEP_BY_STEP_GUIDE.md - Troubleshooting section
2. Check: Backend URL is `http://10.0.2.2:5000`
3. Check: Google Client ID is set
4. Check: Gradle synced successfully

### "Cannot sign in"
1. Check: Backend is running (`python app.py`)
2. Check: Google Client ID is correct
3. Check: Internet connection
4. See SETUP.md - Common Issues

### "Cannot find file/error"
1. Check: You're in correct directory
2. Check: File names match exactly
3. Check: build.gradle.kts syntax is correct
4. See STEP_BY_STEP_GUIDE.md - Troubleshooting

### "Gradle won't sync"
1. Check: No syntax errors in build.gradle.kts
2. Try: File > Sync Now (again)
3. Try: Build > Clean Project
4. See SETUP.md - Advanced Configuration
